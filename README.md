OPS is inspired by popular music software such as **FL Studio, Ableton, GarageBand, and BandLab**. Unlike these, it focuses on **simplicity, speed, and cross-platform usability**.

Unlike most web apps, the interface is built **almost entirely with HTML Canvas** — meaning **very few DOM elements** and a **custom UI engine** designed specifically for audio production.

# How to Use

## Basic Workflow

- Click `inside a track` to create a **pattern**
- Long press `inside a track` (not on a pattern) to open its menu  
  *To Create (Automation Clip, Wave File — still in development)*
- Click a `pattern` to open it in the **Piano Roll**
- Click anywhere in the `Piano Roll` to place notes
- Click `Play` to start the song
- Long press a `Channel` to open its menu (e.g., rename or change color). The same works for `patterns`.

# Keyboard & Mouse Controls

### General

<div align="center">
  
| Key | Action |
|----|----|
| **Space** | Play / Stop |
| **Backspace** or **Z** | Return to start |
  
</div>

### Piano Roll

<div align="center">
  
| Control | Action |
|----|----|
| **Click** | Place note |
| **V + Scroll** | Adjust note volume |
| **Middle Mouse Click** | Select notes |
| **Alt + Scroll** | Scroll horizontally |
| **Shift + Scroll** | Zoom Y |
| **Ctrl + Scroll** | Zoom X |

</div>

# Mobile (Touch Screen)
<div align="center">
  
| Gesture | Action |
