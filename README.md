# Open-Sound
OPS is a free open-source DAW written in JavaScript. Similar to popular DAWs like FL Studio, GarageBand, Ableton, and BandLab, it is designed to be simple to use and cross-platform. The interface is built almost entirely with Canvas rather than DOM elements. This is my largest project.
